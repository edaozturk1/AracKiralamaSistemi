# 🚗 Araç Kiralama Yönetim Sistemi

Bu proje, Python dili ile geliştirilmiş, temel araç kiralama sürecini yönetmeyi sağlayan bir sistemdir. Kullanıcı dostu bir arayüz ile araç ekleme, kiralama gibi işlemleri kolaylaştırmayı amaçlar.

## ✨ Özellikler
*   **Araç Yönetimi:** Yeni araç ekleme, silme ve mevcut araç bilgilerini düzenleme[cite: 1].
*   **Kiralama Sistemi:** Müsait araçları kiralama ve kiradaki araçları iade süreçlerinin takibi[cite: 1].
*   **Veri Yönetimi:** Araç verilerinin `araclar.json` dosyasında kalıcı olarak saklanması[cite: 1].
*   **Gelişmiş Mimari:** Proje; `main.py`, `gui.py` ve `backend/` yapısı ile modüler şekilde tasarlanmıştır[cite: 1].
