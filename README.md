# 🚗 Araç Kiralama Yönetim Sistemi

Bu proje, Python dili ile geliştirilmiş, temel araç kiralama sürecini yönetmeyi sağlayan bir sistemdir. Kullanıcı dostu bir arayüz ile araç ekleme, kiralama gibi işlemleri kolaylaştırmayı amaçlar.

## ✨ Özellikler
*   **Araç Yönetimi:** Yeni araç ekleme, silme ve mevcut araç bilgilerini düzenleme.
*   **Kiralama Sistemi:** Müsait araçları kiralama ve kiradaki araçları iade süreçlerinin takibi.
*   **Veri Yönetimi:** Araç verilerinin `araclar.json` dosyasında kalıcı olarak saklanması.
*   **Gelişmiş Mimari:** Proje; `main.py`, `gui.py` ve `backend/` yapısı ile modüler şekilde tasarlanmıştır.

## 🛠️ Kullanılan Teknolojiler
*   **Dil:** Python
*   **Veri Depolama:** JSON
*   **Geliştirme Ortamı:** PyCharm
*   **Versiyon Kontrol:** Git & GitHub
